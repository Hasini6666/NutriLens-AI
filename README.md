# NutriLens AI 🥗

Smart Food Packet Ingredient Analyzer using OCR

## About the Project

NutriLens AI is a smart web application that analyzes packaged food ingredients and predicts whether a product is healthy, moderate, or harmful.

Users can upload a food packet image, extract ingredient text using OCR (Optical Character Recognition), and receive a health score with risk analysis.

## Features

✅ Upload food packet image
✅ OCR-based ingredient extraction
✅ Ingredient risk detection
✅ Health score system (1–10)
✅ Healthy / Moderate / Harmful status
✅ Recommendation system
✅ Premium interactive UI

## Technologies Used

* HTML
* CSS
* JavaScript
* Tesseract.js (OCR)

## How It Works

1. Upload a food packet image
2. Scan ingredients using OCR
3. Analyze harmful ingredients
4. Generate a health score
5. Show recommendations

## Sample Ingredients Tested

Sugar, Palm Oil, Salt, Sodium Benzoate

Oats, Almonds, Milk

INS 211, E102, Artificial Flavor
